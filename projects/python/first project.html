Program for finding prime numbers

A prime number checker program is a simple tool that helps determine whether a number entered by the user is prime or not. It works by evaluating the number's divisibility; a prime number is only divisible by 1 and itself. The program typically asks the user to input a number, then checks if it has any divisors other than 1 and the number itself, starting from 2 up to the number written by a user. If it finds any such divisors, the program will confirm that the number is not prime; otherwise, it will indicate that the number is prime. This program is useful for quickly verifying prime numbers, especially for beginners learning about number theory.

Where the idea came from

The idea for this project came from a similar problem we were working on in college using JavaScript. I found the challenge of determining prime numbers interesting and wanted to explore how the same logic could be implemented in Python. Since I was already familiar with the basic concepts from our JavaScript exercises, I thought it would be a great opportunity to practice Python and see how the two languages compare in terms of syntax and problem-solving approaches. This project allowed me to apply what I had learned in a new context while strengthening my understanding of Python fundamentals. You can check out the project on my GitHub repository.
